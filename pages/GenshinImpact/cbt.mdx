# Connecting to *anime game* CBT PS
## Required to play

* [Soggy (thx score)](https://cdn.discordapp.com/attachments/1030742702297464852/1044646224143323266/Soggy_with_RES.rar)
* [CBT 1](https://archive.org/download/genshin-impact-cbt-1/Genshin%20Impact%20CBT1.rar)
* [Visual Studio (2022)](https://visualstudio.microsoft.com/)
* [vcpkg](https://github.com/Microsoft/vcpkg)
* [Python 3 or higher](https://www.python.org/downloads/)

<Callout emoji="⚠️" type="warning">
  In Visual Studio, you must get the "Desktop Development with C++" plugin
</Callout>

## Connecting
1. Download soggy from [here](./cbt#required-to-play)
Install dependencies: [vcpkg](./cbt#required-to-play)\
2. Open your terminal on vcpkg and type:
```sh
vcpkg install protobuf lua[cpp]:x64-windows
```

3. Run soggy.exe and dispatch.py (python dispatch.py)
