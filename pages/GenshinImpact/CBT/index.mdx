## Connecting
1. Clone or download soggy and soggy_resources and install dependencies (vcpkg)
2. Open your terminal on vcpkg and type in:
```sh
vcpkg install protobuf lua[cpp]:x64-windows
```

### Building soggy
1. Open soggy folder with Visual Studio
2. Build soggy.exe - Output folder will be in .\out\build\x{'{Build-Config}'}
3. Move soggy_resources to the output folder and rename it to resources
4. Run soggy.exe and dispatch.py (using "python dispatch.py")
5. Run the game

            oS.oRequest.headers.UriScheme = "http";

    }
