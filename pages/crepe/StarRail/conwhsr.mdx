## Connecting

1. Once you are done downloading everything, run "CrepePlusManager.bat". Select the "Install NPM modules" option. (Select this option by typing "3" in the console. Also, you only need to do this once.)
2. After installing, select the "Start the CrepePlus server" option in the console. (Type in "1")
3. Open "StarRail.exe" once CrepePlus has started.


To make an account, use
```sh
/account add (account name) (account uid)
example: 
/account create rrryfoo 1000
```
